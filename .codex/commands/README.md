# Lab04 Commands

Prompt commands in this folder are tuned for the Lab04 CPG Streaming project.

- `/sdd/specify`, `/sdd/plan`, `/sdd/tasks`, `/sdd/implement`: small
  spec-driven workflow for parser/runtime/store/replay/evidence changes.
- `/evidence-check`: audit notebooks, screenshots, and book readiness.
- `/replay-check`: audit idempotent replay proof.
- `/architecture-review`: review pipeline boundaries.
- `/pr-review` and `/agent-team-review`: review diffs for Lab04 risk.
- `/onboard`: build or refresh context.
- `/docs-update`: update Lab04 docs and book content.
